# CODING.md — Lightway

## Build & Development

```bash
npm run build    # Vite production build
npm run dev      # Local dev server
```

## Branch Model

- PRs target `main`
- `main` auto-deploys to Cloudflare Pages

## Process

1. Work in feature branches
2. Run `npm run build` to verify
3. PR to `main`, merge when ready
