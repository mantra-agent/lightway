# Lightway — Mantra Landing Pages

Static marketing site for Mantra. Simple Vite + React + TypeScript.

## Tech Stack

- **Framework:** React 18 + Vite
- **Styling:** Inline styles (no Tailwind)
- **Hosting:** Cloudflare Pages (auto-deploy from `main`)
- **Repo:** github.com/mantra-agent/lightway

## Architecture

Single-page app with client-side routing. All pages are React components under `src/`. Entry point is `src/main.tsx` → `src/App.tsx`.

## Brand

- Black background (#000)
- Mantra logo at `public/mantra-logo.png` or repo-local equivalent
- Clean, minimal, brand-forward design
